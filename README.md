# 🚢 Titanic Dataset - Exploratory Data Analysis (EDA)

This project focuses on performing Exploratory Data Analysis (EDA) on the Titanic Machine Learning dataset. The goal is to understand the features affecting passenger survival and prepare the dataset for further modeling.

The dataset contains information about passengers such as age, sex, ticket class, fare, and whether they survived. Through this analysis, we explore patterns and relationships between variables to gain deeper insights.

---

## 📊 Key Tasks Performed

1. Displayed the top 5 and last 3 rows of the dataset
2. Found the shape of the dataset (rows and columns)
3. Retrieved data types, memory usage, and structure of the dataset
4. Generated overall statistics of the dataframe
5. Filtered data using conditional operations
6. Checked for and handled missing values
7. Dropped unnecessary columns to simplify analysis
8. Encoded categorical variables into numerical values
9. Performed univariate analysis:
10. Counted how many people survived vs. died
11. Analyzed class distribution (1st, 2nd, 3rd)
12. Counted number of male and female passengers
13. Performed bivariate analysis:
14. Compared survival rates based on gender
15. Compared survival rates based on passenger class
16. Applied basic feature engineering to enhance the dataset

## 📂 Dataset Overview

The Titanic dataset includes:
- Passenger ID
- Name, Sex, and Age
- Ticket Class (Pclass)
- Fare and Embarked Port
- Number of siblings/spouses aboard
- Number of parents/children aboard
- Survival status (0 = No, 1 = Yes)
